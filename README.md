# PisciProject 🐟

Sistema web para verificação de compatibilidade entre peças de computador.

---

## 📌 Objetivo

Este projeto tem como objetivo desenvolver um site capaz de verificar a compatibilidade entre diferentes componentes de hardware, como:

- Processador (CPU)
- Placa-mãe
- Memória RAM
- Outros componentes

O sistema permitirá ao usuário selecionar peças e receber um retorno informando se elas são compatíveis entre si.

---
